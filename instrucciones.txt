Se le contacta para realizar una página web de bicicletas(puede ser un tienda, una marca en específico, tipo, o cualquier tema relacionado con bicicletas), para lo cual, usted decide realizar un Landing Page para desarrollar el trabajo.
En esta situación el cliente le solicita que tenga ciertos requerimientos. 

•	Menú de navegación a las diferentes secciones. (20 puntos)

•	Agregar Home o header con su respectivo texto centrado y un color que resalte, el texto debe ser su nombre y apellido. (10 puntos)

•	Sección con ofertas (con algún recuadro bien definido), que contenga icono o foto, titulo descriptivo y valor (al menos 3 recuadros alineados en la misma fila y centrados, con margen y espaciado). (15 puntos)

•	Crear galería que muestre al menos 4 imágenes. (25 puntos)

•	Crear Footer que contenga datos de contacto del creador (fono, email, dirección, link al repositorio de github.). (15 puntos) 

•	Utiliza github para guardar el trabajo. (15 puntos)
